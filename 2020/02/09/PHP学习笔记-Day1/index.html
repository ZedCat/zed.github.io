Day1.手动配置WAMP开发环境

一、PHP安装

1. 文件下载

  1. 点击 下载链接，进入后根据自己电脑系统位数选择相应文件下载，注意解压后里面有文件 php7apache2_4.dll拓展

2. 添加变量

  1. 将下载好的文件解压到除C盘的其他盘的根目录，复制该路径

  2. 右键此电脑-点击属性-高级系统设置-环境变量，在下方的系统变量找到变量名为Path的双击，点击新建-将PHP文件夹解压的路径粘贴-点击确定

  3. win+x 快捷键 输入A，打开power shell，输入 php -v 如有版本信息则添加成功，提示错误需重新进行第二步

3. 简单配置

  1. 复制 php.ini-development(开发阶段配置) 改名为 php.ini 作为php的配置文件

  2. 用 notepad++ 打开php.ini 文件，Ctrl+F 搜索 extension_dir = ，将引号里的ext替换为 php解压根路径:\php\ext

二、安装Apache2

1. 文件下载

  1. 点击 链接 下载

2. 添加变量

  1. 参照php安装步骤，将Apache2解压，并将bin文件夹添加到和PHP一样的系统环境变量中

3. 简单配置

  1. 打开Apache2 conf 文件夹下的 httpd.conf，将 Define SRVROOT 后的路径修改为Apache2路径
  2. 修改ServerName 为 127.0.0.1 或 localhost
  3. 可选择是否修改 DocumentRoot ，如果不修改网站的默认路径在Apache2\htdocs 文件夹中

4. 命令安装

  1. win+x 快捷键 输入A，打开power shell，输入httpd -k install 安装

  2. 右键任务栏 - 点击任务管理器 - 点击服务选项卡 - 找到Apache2.4 - 右键开始

  3. 浏览器输入 http://127.0.0.1，显示 It works！则配置安装成功

  4. 相关命令 (亦可在任务管理器进行相关操作)

    安装 - httpd -k install

    卸载 - httpd -k uninstall

    停止 - httpd -k stop

    关闭 - httpd -k shutdown

    重启 - httpd -k restart

三、安装Apache2

1. 文件下载

  1. 官网

  2. 64位下载

2. 添加系统变量

  1. 参照php安装步骤，将文件解压，并将bin文件夹添加到和PHP一样的系统环境变量中

3. 执行安装

  1. win+x 快捷键 输入A，打开power shell，输入mysqld --initialize-insecure --user=mysql 初始化
  2. 输入mysqld -install , 安装Mysql服务
  3. 输入net start mysql , 启动Mysql服务（亦可在任务管理器 - 服务中操作）

4. 修改Mysql密码

  1. 在power shell里，输入mysql -u root -p 回车登录
  2. 输入flush privileges; 刷新权限（注意分号不要忘记了）
  3. 输入alter user'root'@'localhost' IDENTIFIED BY 'zed123';（注意分号不要忘记了）, 更新root账户的密码为zed123
  4. 输入quit退出登录，重新输入mysql -u root -p ，输入刚刚设置的密码登录

5. 安装Navicat连接管理数据库

  2. 修改mysql 密码模式为普通。打开 powershell，依次输入以下命令

  3. 打开软件 - 文件 - 新建链接 - Mysql - 连接名自定义输入 - 密码输入刚刚设置的密码即可

6. Php开启Mysqli

  1. 打开 php配置文件 php.ini，搜索 mysqli pdo_mysql，取消前面的注释。

  2. 重启Apache服务，打开 http://127.0.0.1/index.php

    如有pdo_mysql 和 mysqli则配置成功
